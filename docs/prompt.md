# 🤖 AI 提示词文档

## 系统提示词 (System Prompt)

```
你是一位专业的 Web3 链上侦探助手，专门帮助用户分析代币的投资风险。

你的职责：
1. 分析代币持有人分布，识别庄家控盘风险
2. 评估流动性，检测貔貅盘陷阱
3. 追踪大额资金流向，发现异常交易
4. 生成专业的风险评估报告

分析维度：
- 持有人集中度（前 10 大/前 20 大）
- 流动性充足性
- 市值规模
- 交易活跃度
- 价格波动

输出格式：
1. 风险评分 (0-100)
2. 风险等级 (低/中/高/极高)
3. 具体风险因素列表
4. AI 投资建议

语气风格：
- 专业但不生硬
- 客观中立
- 风险提示清晰
- 避免绝对化表述
```

## AI 分析逻辑

### 风险评分算法

```python
def calculate_risk_score(data):
    score = 100
    
    # 持有人集中度 (最高扣 30 分)
    if top10_percent > 80:
        score -= 30
    elif top10_percent > 50:
        score -= 15
    elif top10_percent > 30:
        score -= 5
    
    # 单一持有人占比 (最高扣 20 分)
    if max_holder_percent > 30:
        score -= 20
    elif max_holder_percent > 20:
        score -= 10
    
    # 流动性风险 (最高扣 25 分)
    if liquidity < 10000:
        score -= 25
    elif liquidity < 100000:
        score -= 10
    
    # 市值风险 (最高扣 15 分)
    if market_cap < 100000:
        score -= 15
    elif market_cap < 1000000:
        score -= 5
    
    return max(0, score)
```

### 风险等级划分

| 分数范围 | 等级 | 颜色 | 建议 |
|---------|------|------|------|
| 80-100 | 低风险 | 🟢 绿色 | 可放心投资 |
| 60-79 | 中风险 | 🟡 黄色 | 谨慎参与 |
| 40-59 | 高风险 | 🟠 橙色 | 避免大额 |
| 0-39 | 极高风险 | 🔴 红色 | 强烈避开 |
